# WebSocket TODO App with Redis

Проект реализует WebSocket сервер и клиент с CRUD-операциями для задач (TODO), используя Redis в качестве хранилища данных.

---

## 🚀 Технологии

- :contentReference[oaicite:0]{index=0} — серверная среда выполнения JavaScript  
- :contentReference[oaicite:1]{index=1} (ws) — обмен сообщениями в реальном времени  
- :contentReference[oaicite:2]{index=2} — in-memory база данных для хранения задач  
- Docker (опционально) — для запуска Redis
